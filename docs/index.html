Spin up the tool and an isolated Tor proxy with a single docker-compose command.

Use the --tor flag to route traffic through SOCKS5, preventing IP bans and staying hidden.

Get instant notifications on Discord or Telegram when target leaks are discovered.

📊 HTML Reporting

Generate pristine, client-ready HTML reports detailing the exact source and snippet of the leak.

🛠️ Installation

Option 1: Docker (Recommended)

The cleanest way to run the tool without polluting your local environment.

# Clone the repository
git clone https://github.com/rokibulroni/databreach-finder.git
cd databreach-finder

# Start the Tor Proxy in the background
docker-compose up -d tor

# Run the Databreach Finder
docker-compose run databreach-finder -q "target_company" --extract emails --tor

Option 2: Native Python

Ensure you have Python 3.11+ installed on your system.

git clone https://github.com/rokibulroni/databreach-finder.git
cd databreach-finder

# Create and activate a virtual environment
python3 -m venv venv
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

💻 Usage Guide

The tool comes with a beautiful CLI interface. Here are some common ways to use it:

1. Basic Search

Hunt for a keyword and extract any lines containing it:

python3 databreach_finder.py --query "example.com"

2. Targeted Regex Extraction

Find specific data patterns related to your target using the --extract flag. Supported options: emails, cards, ipv4, btc, eth, aws_key, jwt, rsa_private.

python3 databreach_finder.py --query "company dump" --extract aws_key

3. Stealth Mode (Tor Routing)

Route all your requests through the Tor network. Requires Tor running locally on port 9050, or via Docker.

python3 databreach_finder.py --query "target" --extract btc --tor

4. Generate HTML Report

Generate a professional HTML report alongside the text output for clients.

python3 databreach_finder.py --query "example.com" --extract emails --report html

🧠 Advanced Configuration

Databreach Finder Pro supports zero-code customization via a config.json file.

  1. Rename config.json.example to config.json.
  2. Add Custom Paste Sites: Append specific deep web or surface web URLs.
  3. Add Custom Regex Patterns: Hunt for your own proprietary keys.
  4. Configure Webhooks: Add your Discord Webhook URL or Telegram Bot Token for real-time alerting.
